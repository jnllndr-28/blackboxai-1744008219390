# Geologic Timescale Quiz

An interactive quiz game about Earth's geologic history.

## Features

- 10 multiple-choice questions about geologic periods and events
- Responsive design works on mobile and desktop
- Instant feedback on answers
- Detailed results with explanations
- Attractive geology-themed design

## How to Play

1. Open `index.html` in a web browser
2. Click "Start Quiz"
3. Answer all 10 questions
4. View your results and learn from the explanations
5. Try again to improve your score!

## Technical Details

- Built with HTML, CSS, and JavaScript
- Uses Tailwind CSS for styling
- Questions stored in JSON format
- No backend required - runs entirely in the browser

## Questions

The quiz covers:
- Major geologic eras (Paleozoic, Mesozoic, Cenozoic)
- Key events in Earth's history
- Evolutionary milestones
- Climate changes and extinctions
